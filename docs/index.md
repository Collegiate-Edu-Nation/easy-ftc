# Welcome to the docs for easy-ftc

## Overview
[easy-ftc](https://github.com/Collegiate-Edu-Nation/easy-ftc) is a library for easily leveraging in-the-box FTC mechanisms and sensors, including

* Arm
* Claw
* Drivetrain
* Lift
* Color Sensor
* Distance Sensor
* Touch Sensor
